# Next.js Task UI Builder

Builds responsive Todo interface with Tailwind and protected routes for Todo application.

## Version
1.0

## Category
Frontend

## Inputs

| Name | Type | Description | Default |
|-------|------|-------------|----------|
| ui_spec | string | Path to UI specification | `@specs/ui/pages.md` |

## Outputs

- Task list page
- Add task form
- Task card component
- Protected layout wrapper

## Instructions

Create modern Todo UI with the following features:

- Protected routes with auth redirect
- Responsive task list with status indicators
- Add task modal/form
- Edit/delete actions
- Loading and error states
- Tailwind CSS styling

## Example Usage

```
@skills/nextjs/nextjs-task-ui.md build main interface
```
