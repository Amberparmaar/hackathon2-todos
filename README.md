# Evolution of Todo

**Hackathon II - Panaversity**

A progressive 5-phase Todo application evolution, demonstrating modern software development practices from simple console app to fully cloud-native, event-driven, AI-powered distributed system.

## Quick Start

```bash
# Initialize project (if not already done)
uv init --name texthackathon-todo

# Read the constitution
cat .specify/memory/constitution.md

# Start with Phase 1
cd phase1
```

## Project Overview

This project evolves a Todo application through 5 sequential phases:

| Phase | Name | Complexity | Tech Stack | Features |
|-------|------|------------|------------|----------|
| I | In-Memory Python Console App | Low | Pure Python | Basic CRUD, Console UI |
| II | Full-Stack Web App | Medium | Next.js + FastAPI + Neon | CRUD + Auth + Persistence |
| III | AI-Powered Chatbot | Medium-High | OpenAI ChatKit + Agents SDK + MCP | Natural Language Interface |
| IV | Local Kubernetes Deployment | High | Docker + Minikube + Helm | Containerized Deployment |
| V | Advanced Cloud Deployment | Very High | Dapr + Kafka + Cloud K8s | Event-Driven + Advanced Features |

## Core Principles

1. **Spec-Driven Development** - Every feature starts with a written specification
2. **Zero Manual Coding** - Code generated by Claude Code only
3. **Phase-by-Phase Progression** - Each phase builds on the previous
4. **Reusable Intelligence** - Use agent skills and subagents
5. **Strict Tech Stack Compliance** - Only specified technologies per phase
6. **Security & Best Practices** - User isolation, stateless design, clean code
7. **Monorepo Structure** - Phase isolation in folders
8. **Documentation Excellence** - Complete documentation at each phase

## Required Features

### Basic (All Phases)
- Add Task (title + optional description)
- Delete Task
- Update Task
- View Task List
- Mark as Complete / Incomplete

### Intermediate (Phase V)
- Priorities (high/medium/low)
- Tags/Categories
- Search by keyword
- Filter by status, priority, date
- Sort by due date, priority, title

### Advanced (Phase V)
- Recurring Tasks (e.g., weekly, daily)
- Due Dates & Time Reminders

### AI Integration (Phases III–V)
- Natural language interface to manage all tasks

## Bonus Features (+600 Total Points)

- **Reusable Intelligence** via subagents & skills (+200)
- **Cloud-Native Blueprints** via Agent Skills (+200)
- **Urdu language support** in chatbot (+100)
- **Voice input/commands** (+200)

## Workflow

For each phase, follow this strict workflow:

```
Specify (specify.md)
→ Plan (plan.md)
→ Tasks (tasks.md)
→ Implement (implement.md)
```

**Rule**: No code generation without a referenced Task ID.
